Privacy Policy of KomunDroid

KomunDroid is a non-profit utility application that does not request/process/store any personal user information, except for the data, that user enters manually in the application.



Application stores all the data on the device using standard sqlite database, provided by default by Android platform.



It requests following permissions:
android.permission.CAMERA     This permission is required in order to use built-in flash. It is useful to use it as a source of light in dark places, to lighten the meters, for example.-



There are no hidden or visible Ads in the application, so the device and the user information are never exposed through the app itself or through embedded 3rd party libraries.



If you have any questions, please contact the developer.
